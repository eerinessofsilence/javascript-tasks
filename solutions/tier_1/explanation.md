# Пояснения к решениям (tier_1)

Ниже кратко объяснена логика каждого решения из `solutions/tier_1/solutions.js`.

## 1 уровень

### 1. Возведение в степень (`power`)

- Используется оператор `**`.
- Формула: `A ** B`.

### 2. День недели (`getWeekDay`)

- Дни недели хранятся в массиве.
- Пользователь вводит число от `1` до `7`, поэтому берется индекс `dayNumber - 1`.
- Добавлена базовая проверка диапазона и типа.

### 3. Большее из двух (`maxOfTwo`)

- Сравнение через тернарный оператор: `a > b ? a : b`.
- Возвращается большее число.

### 4. Доллары в гривны (`usdToUah`)

- Простое умножение: `usd * rate`.
- `rate` - курс одной единицы доллара в гривне.

## 2 уровень

### 1. Четность числа (`parity`)

- Проверка через остаток от деления на 2: `number % 2`.
- Если остаток `0`, число чётное.

### 2. Минимум из трёх (`minOfThree`)

- Используется банальный перебор чисел "больше / меньше" и тернарные операторы.

### 3. Режим сна (`sleepStatus`)

- Логика по условиям:
  - `H < A` -> `"Недосып"`
  - `H > B` -> `"Пересып"`
  - иначе -> `"Это нормально"`
- Проверка `A <= B` не нужна, это гарантируется условием.

## 3 уровень

### 1. Калькулятор (`calculator`)

- Используется `switch` по строке операции.
- Поддержаны операции: `+`, `-`, `*`, `/`, `mod`, `pow`, `div`.
- Для `/`, `mod`, `div` добавлена защита от деления на ноль.
- Для `div` используется `Math.trunc(first / second)` - целая часть при делении.

### 2. Делимость чисел (`divisibilityCheck`)

- Проверяются два независимых факта:
  - делится ли `A` на `B` (`a % b === 0`)
  - делится ли `B` на `A` (`b % a === 0`)
- Возвращается объект с двумя булевыми полями.
- Если делитель равен нулю, соответствующая проверка считается `false`.

### 3. Одинаковые цифры в трёхзначном числе (`hasSameDigits`)

- Берётся модуль числа (`Math.abs`), чтобы корректно работать с отрицательными.
- Извлекаются цифры:
  - сотни: `Math.floor(number / 100)`
  - десятки: `Math.floor(number / 10) % 10`
  - единицы: `number % 10`
- Возвращается `true`, если хотя бы две цифры совпадают.
- Есть проверка на трёхзначность числа.
