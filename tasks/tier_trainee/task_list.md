## 1 уровень

1. Пользователь вводит три числа `A`, `B`, `C`. Выведите результат выражения `A + B * C` и отдельно результат `(A + B) * C`.

2. Создайте переменные `userName`, `userAge`, `userCity`. Сформируйте строку: `"Имя: ..., возраст: ..., город: ..."` и выведите её.

3. Для значений `42`, `"42"`, `true`, `null`, `undefined`, `{}` выведите тип каждого значения через `typeof`.

4. Пользователь вводит два числа в виде строк. Преобразуйте их в числа и выведите сумму. Если хотя бы одно значение нельзя преобразовать, выведите `"Некорректный ввод"`.

5. Дан счётчик `counter`. Выполните операции: `counter += 5`, затем `counter *= 2`, затем `counter--`. Выведите итог.

6. Даны числа `A` и `B`. Увеличьте `A` на `B` через `+=`, после этого уменьшите результат на `1` унарным оператором и выведите значение.

## 2 уровень

1. Создайте объект `user` с полями `name`, `age`. Добавьте поле `isStudent`, добавьте поле `email`, затем удалите поле `age`.

2. Дан объект пользователя. Добавьте вложенные свойства `user.address.city` и `user.address.zip`.

3. Даны переменные `title`, `price`, `inStock`. Создайте объект товара в сокращённом формате записи свойств.

4. Для объекта банковского счёта реализуйте методы `deposit` и `withdraw`, использующие `this`.

5. Дан массив чисел. С помощью глобального объекта `Math` найдите минимум, максимум и округлённое среднее.

6. Преобразуйте объект в JSON-строку и затем восстановите объект обратно.

## 3 уровень

1. Напишите функцию, которая изменяет имя пользователя в переданном объекте. Покажите, что исходный объект мутируется.

2. Напишите версию из задачи 1 без мутации: функция должна возвращать новый объект с обновлённым именем.

3. Напишите функцию приветствия с параметром по умолчанию (если имя не передано, использовать `"Гость"`).

4. Напишите функцию `applyOperation(a, b, callback)`, которая вызывает переданный колбэк для двух чисел.

5. Реализуйте пример цепочки областей видимости через вложенные функции, где внутренняя функция использует значения из внешних областей.

6. Реализуйте калькулятор на `switch` в строгом режиме. Поддержите операции: `+`, `-`, `*`, `/`, `mod`, `pow`, `div`.
