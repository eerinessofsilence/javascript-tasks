# Сборник задач по JavaScript

Репозиторий с практическими задачами для отработки базовых навыков JavaScript.

## Структура проекта

```text
tasks/
  tier_*/
    task_list.md   # список задач по уровням
    task.js        # шаблон для решений

solutions/
  tier_*/
    solution.js    # готовые решения
    explanation.md # пояснения к решениям
```

## Быстрый старт

1. Убедитесь, что установлен Node.js.
2. Откройте список задач: `tasks/tier_*/task_list.md`.
3. Выберите уровень и номер задачи.
4. Запишите решение в `tasks/tier_*/task.js`.
5. Запустите файл:

```bash
cd tasks/tier_*
node task.js
```

## Как решать задачи

1. Сначала прочитайте условие в `tasks/tier_*/task_list.md`.
2. Решайте задачу в `tasks/tier_*/task.js` в соответствующем блоке уровня.
3. Проверяйте результат через `console.log` и запуск `node task.js`.
4. После завершения переходите к следующей задаче.

## Рекомендации

- Выбирайте задачи по своему текущему уровню.
- Старайтесь сначала решить самостоятельно, затем сравнить с `solutions/tier_*/solution.js`.
- Если смотрите готовое решение, разберите, почему оно работает, и повторите без подсказок.
